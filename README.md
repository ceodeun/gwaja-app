# 🍪 과자 (과외하자)

> 크래커처럼 바삭한 선생님, 찹쌀떡처럼 찰떡인 매칭

## 페이지

| URL | 설명 |
|-----|------|
| `/` | 메인 앱 (학생/선생님/학부모) |
| `/map` | 전국 학교 지도 |
| `/location` | 과외 장소 찾기 + 거리 계산 |

## 기능

- 🎒 **학생** — 선생님 검색/필터, 수업 일정, 채팅, 학습 목표, 성적 그래프, 후기 작성
- 🎓 **선생님** — 대화 요청, 스케줄 메이커, 정산 내역, 선호 이동거리 설정
- 👨‍👩‍👧 **학부모** — 수업 현황, 수업료 확인, 인수인계 히스토리
- 🗺️ **지도** — 전국 대학/중고등학교, 과외 장소, 이동시간 계산
